# Architecture Decision Records (ADRs)

This directory contains architecture decision records for Actifix. ADRs capture significant decisions, the context, and consequences.

## ADR format
```markdown
# ADR-XXXX: Title

## Status
[Proposed | Accepted | Deprecated | Superseded by ADR-YYYY]

## Context
[Why this decision is needed]

## Decision
[What was decided]

## Consequences
[What gets easier/harder]

## Alternatives Considered
[Options and why they were rejected]

## References
[Links or related docs]
```

4. Cross-reference related docs in `docs/`.
